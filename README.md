&nbsp;&nbsp;&nbsp; A Clipboard Manager desktop app made for Mac OS that lives in the menu bar

<hr>

To install source code and run app:

$ git clone https://github.com/kieranmakes/Espresso-Board/

$ npm install

$ npm start

<hr>

run ***CMD + Q*** to exit app
